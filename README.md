# Class Game - Parkour Challenge 🎮

A fast-paced, reaction-time parkour game with progressive difficulty and dynamic scoring mechanics.

## 🎯 Features

### Core Gameplay
- **Speed Scaling** - Game increases in difficulty every 2 seconds
- **Survival Bonus** - +50 points at 5-second mark
- **Timed Rewards** - +50 points every 10 seconds (Normal+ tiers)
- **Dynamic Color Scheme** - Colors change rapidly every 0.5 seconds
- **Parkour Mechanics** - Jump-based movement with collision detection
- **Real-time HUD** - Live score, timer, speed, and status displays
- **Milestone Tracking** - Visual indicators at 5s, 10s, 15s, 20s, 25s, 30s

### Difficulty Tiers
| Tier | Entry Fee | Start Score | Win Reward |
|------|-----------|-------------|------------|
| Easy | Free | 50 | - |
| Normal | Free | 50 | - |
| Hard | Free | 100 | +300 |
| Hell | 200 | 0 | +500 |

## 🎮 How to Play

1. **Select Difficulty** - Choose from 5 difficulty levels
2. **Jump & Move** - Use keyboard or touch controls to navigate platforms
3. **Avoid Obstacles** - Survive by dodging enemy spikes
4. **Survive Longer** - The longer you survive, the higher your score
5. **Reach Checkpoints** - Hit milestones to maximize score bonuses

### Controls
- **Keyboard**: Arrow keys or WASD to move, Space/W to jump, ESC to pause
- **Mobile**: Swipe left/right to move, swipe up to jump

## 📊 Scoring

- **Easy**: Start at 50, single +50 bonus available (max 100)
- **Normal**: Start at 50, +50 every 10s (max 200)
- **Hard**: Start at 100, must survive 30s to win +300
- **Hell**: Must survive 30s to win +500
- **Impossible**: Must survive 30s to win +1000
